# Trial Studios Coding Workshop

A project-based learning initiative by Trial Studios.

This workshop is designed for aspiring developers who want to learn by building real projects instead of relying solely on notes and theory. Participants complete practical challenges, submit their work, and gradually develop real-world development skills.

---

## Philosophy

We believe the best way to learn programming is by creating things.

Instead of memorizing syntax, participants are encouraged to:

* Build projects
* Experiment with code
* Research solutions
* Learn from mistakes
* Develop problem-solving skills

Using documentation, search engines, tutorials, and AI assistants as learning tools is allowed and encouraged. The goal is understanding, not memorization.

---

## Current Program

### Web Development Track

Participants will receive projects of increasing difficulty.

Projects may cover:

* HTML
* CSS
* JavaScript
* Responsive Design
* Git & GitHub
* Frontend Development
* Real-World Project Structure

---

## Rules

### Allowed

* Documentation
* Tutorials
* Internet Research
* AI Assistants
* Experimentation

### Not Allowed

* Copying another participant's work
* Submitting projects you do not understand
* Reusing complete templates without modification

Participants may be asked to explain parts of their code.

---

## Submission

Submit your project using:

* GitHub Repository Link
* GitHub Pages Link (optional)

Late submissions may still be accepted, but completing projects before the deadline is strongly encouraged.

---

## Beginner Friendly

No prior experience is required.

If this is your first time writing code, that's completely fine.

Start small, learn consistently, and focus on completing projects.
